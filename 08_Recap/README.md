## What's Covered

The recap traces the "MISSING chain" — what each session added that the previous one lacked:

| Session | Key Concept | What Was Missing → Next Session |
|---------|-------------|-------------------------------|
| S00 | TF-IDF search | Words counted, no meaning |
| S01 | Embeddings (BoW → FastText) | Fixed vectors, no context |
| S02 | Transformer (self-attention) | Architecture only, no GPT story |
| S03 | GPT-1/2/3 + Alignment | Theory but can't run LLMs |
| S04 | BPE + Sampling params | Understand internals, can't use LLMs |
| S05 | Local LLMs + API providers | Can call LLMs but prompts are manual |
| S06 | DSPy prompt optimization | Optimized prompts but no unified framework |
| S07 | LangChain | Unified interface for everything |
