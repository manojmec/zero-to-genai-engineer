# Product Overview

**Zero to GenAI Engineer** is a modular, open-source educational course that takes learners from complete beginners to production-ready GenAI engineers.

## Purpose
- Structured curriculum covering search engines, embeddings, transformers, GPT architecture, tokenization, local LLMs, prompt engineering, agentic coding, RAG, agents, and deployment
- Each session answers "what was missing from the last one?" forming a progressive learning chain
- Real projects built during the course (movie recommender, Distill assessment tool, stock scanner, Holmes GPT)

## Audience
- Aspiring AI/ML engineers learning GenAI from scratch
- Professionals upskilling from traditional ML to generative AI
- Contributors to open-source projects built during the course (e.g., Distill)

## Key Principles
- Theory paired with hands-on code — every session produces a working artifact
- Notebooks are the primary teaching medium (Jupyter)
- Full-stack demo apps accompany key sessions (FastAPI backend + React frontend)
- Content released incrementally (weekly sessions)
